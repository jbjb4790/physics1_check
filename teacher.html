학생 결과를 Google Sheets로 깔끔하게 모으세요

Apps Script 웹 앱 URL을 한 번만 연결하면 회차별 점수, 오개념 태그, 학생 메모가 자동으로 누적됩니다.

Apps Script 코드 열기

Quick start

처음 한 번만 연결하면 됩니다

1Google Sheet 만들기

예: 물리Ⅰ 진단결과라는 새 스프레드시트를 만듭니다.

2Apps Script 열기

시트 메뉴에서 확장 프로그램 → Apps Script를 엽니다.

3Code.gs 붙여넣기

google-apps-script/Code.gs 내용을 붙여넣고 저장합니다.

4웹 앱 배포

배포 → 새 배포 → 웹 앱, 실행 권한은 본인, 액세스는 Anyone으로 설정합니다.

5URL 복사

배포 후 생기는 Web app URL을 복사합니다.

6config.js 수정

SUBMIT_URL에 URL을 붙여넣고 GitHub에 다시 업로드합니다.

config.js

붙여넣을 위치

아래처럼 Web app URL만 바꾸면 됩니다.

window.PHYSICS_CHECK_CONFIG = {
  SUBMIT_URL: "https://script.google.com/macros/s/배포ID/exec",
  CLASS_NAME: "물리Ⅰ",
  AUTO_SUBMIT: true,
  REQUIRE_STUDENT_NAME: true
};
연동 테스트

config.js에 URL을 입력한 뒤 이 버튼을 누르면 테스트 행이 Google Sheet에 추가됩니다.

Apps Script는 요약용 제출결과 시트와 문항 분석용 문항별응답 시트를 자동으로 만듭니다. Drive 폴더 ID를 Code.gs에 입력하면 제출 리포트 txt 파일도 선택적으로 저장됩니다.
