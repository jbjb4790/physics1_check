운영 방식

각 회차 수업 종료 후 아래 링크 중 해당 회차를 학생에게 보내면 됩니다. GitHub Pages에 올린 뒤에는 lesson.html?session=회차번호 형태로 링크가 고정됩니다.

응답 수집 안내

이 버전은 무료 GitHub Pages에서 바로 작동하도록 서버 없이 만들었습니다. 자동 채점과 피드백은 가능하지만, 학생 답안을 중앙 서버에 저장하지 않습니다. 학생은 결과 리포트를 복사하거나 txt 파일로 내려받아 제출할 수 있습니다.
